
# Proxy para conectar PyQt5 con InfinityFree

Este pequeño servidor redirige solicitudes desde tu app de escritorio hacia InfinityFree para evitar el bloqueo.

## Instrucciones:

1. Sube este proyecto a un repositorio en GitHub.
2. Ve a https://render.com y crea una nueva Web Service.
3. Selecciona el repositorio subido.
4. Render detectará automáticamente Flask y lo ejecutará.
5. Usa la URL pública del servicio Render como puente.

Ejemplo:
- https://mi-anidesk-proxy.onrender.com/registrar
- https://mi-anidesk-proxy.onrender.com/obtener
